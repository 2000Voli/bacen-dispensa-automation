\# Overview



This project automates the submission of regulatory dispensation requests inside SCR/BACEN systems using structured Excel inputs.



The solution processes multiple institutions and reporting periods, performs automated form interaction and exports execution results to Excel.



\---



\# Problem Solved



Manual submission of dispensation requests in SCR/BACEN environments was repetitive, operationally inefficient and prone to manual errors.



This automation enabled batch processing of multiple institutions and reporting periods, reducing manual workload and improving operational consistency.
